## 🕹️ O Jogo
O objetivo do jogo é encaixar diferentes tipos de peças em um tabuleiro de dimensões 10x10cu. O programa gerencia um estoque limitado e valida cada movimento.

### Regras Principais:
- **Tabuleiro:** Matriz 10x10.
- **Estoque de Peças:**
  - Tipo 1: 6 unidades
  - Tipo 2: 5 unidades
  - Tipo 3: 7 unidades
  - Tipo 4: 5 unidades
  - Tipo 5: 7 unidades
- **Comandos:** O usuário informa o tipo da peça, linha e coluna.
- **Término:** O jogo acaba ao completar o tabuleiro ou quando o usuário informa `0` para desistir.
